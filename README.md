微信SDK
===
>希望打造一个尽量全面的微信平台管理工具，包含公众号、小程序、微信商户、移动app、网站app等。


当前为方便开发，第一个大版本中所有子扩展均集成在一起，并且主维护 公众号 功能， 后期将会单例拆分各个子扩展。

当前已更新 PHP7.4 类属性强类型，请注意您的PHP版本是否为 PHP7.4.0 及以上
